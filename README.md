## 🚀 Quick Start

### One-Click Demo (Google Colab)
Click the **Open in Colab** badge at the top. No installation. Runs instantly.

### Local Python API
```bash
git clone https://github.com/divineearthly/KAVACH-Sovereign-Intelligence.git
cd KAVACH-Sovereign-Intelligence
pip install -r requirements.txt
python kavach_main.py
```

C++ High-Performance Engine (9.2M events/sec)

```bash
cd cpp-engine
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
make -j$(nproc)
./kavach
```
